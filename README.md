# [第二次初恋](https://hoo.be/xiuxiuww)


他们结婚四十年，拌嘴了四十年。从青菜咸淡吵到孩子教育，像两棵盘根错节的老树，谁也离不开谁，却也互相硌着。

一天清晨，老太太醒来，发现老头忘了她。阿兹海默症像一块橡皮，擦掉了他记忆中关于她的所有笔画。他变回了一个温和、客气的陌生老人。

她不得不重新教他吃饭、穿衣，像照顾一个孩子。他不再和她争吵，只是安静地看着她。一天，她推着轮椅上的他去晒太阳，他忽然拉住她的手，怯生生地，递给她一朵刚摘的野花，眼神清澈得像少年。

“你……真好看。”他羞涩地说。

老太太愣住，泪水涌出。她失去了吵吵闹闹的丈夫，却仿佛迎来了一个羞涩的、追求她的少年郎。

原来，爱有轮回。即使忘记全世界，我依然会凭借本能，再次走向你。

（永恒：纵使记忆消亡，爱你的本能已深入骨髓，成为比记忆更久远的东西）
